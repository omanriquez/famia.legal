Pertanyaan yang Sering Diajukan

Bagaimana cara kerja sinkronisasi iCloud?

Famia menggunakan iCloud untuk menyinkronkan data Anda secara otomatis di semua perangkat Apple Anda. Pastikan iCloud diaktifkan di perangkat Anda dan Anda masuk dengan Apple ID Anda.

Berapa banyak anggota yang dapat saya tambahkan?

Versi gratis Famia memungkinkan Anda menambahkan hingga 3 anggota keluarga. Dengan Famia Pro, Anda dapat menambahkan anggota tanpa batas.

Bagaimana cara mengaktifkan Face ID/Touch ID?

Buka Pengaturan di aplikasi dan aktifkan "Autentikasi Biometrik". Ini akan melindungi informasi keluarga Anda dengan Face ID atau Touch ID.

Bisakah saya mengekspor data saya?

Ya, Anda dapat mengekspor semua data Anda dari bagian Pengaturan. Data diekspor dalam format PDF.

Apakah data saya aman?

Tentu saja. Semua data Anda disimpan hanya di perangkat Anda dan di iCloud pribadi Anda. Famia tidak memiliki akses ke informasi Anda dan kami tidak membagikannya dengan pihak ketiga.

Bagaimana cara kerja uji coba gratis Famia Pro?

Famia Pro menawarkan uji coba gratis selama 7 hari. Selama periode ini, Anda akan memiliki akses penuh ke semua fitur premium. Anda dapat membatalkan kapan saja dari akun Apple ID Anda.

Bisakah saya menggunakan Famia di iPad dan iPhone?

Ya, Famia dioptimalkan untuk iPhone, iPad, dan Mac (Catalyst). Data Anda disinkronkan secara otomatis di semua perangkat Anda.
