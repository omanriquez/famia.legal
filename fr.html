Politique de Confidentialité

Dernière mise à jour: Janvier 2025

Introduction

Chez Famia, votre vie privée est notre priorité. Cette politique explique comment nous gérons vos informations lorsque vous utilisez notre application d'organisation familiale.

Informations que Nous Collectons

Données que VOUS fournissez :

  • Informations sur les membres de la famille (noms, dates de naissance, photos)
  • Informations de santé (allergies, groupe sanguin, médicaments, rendez-vous médicaux)
  • Documents personnels (cartes d'identité, passeports, cartes)
  • Informations financières (dépenses, budgets)
  • Informations sur les véhicules, routines, écoles et animaux

Où Vos Données Sont Stockées

Toutes vos données sont stockées exclusivement dans :

  • Votre appareil - Les données sont enregistrées localement sur votre iPhone ou iPad
  • Votre iCloud personnel - Si iCloud est activé, les données se synchronisent avec votre compte Apple

Famia N'A PAS de serveurs propres. Nous ne collectons pas, ne stockons pas et n'avons pas accès à vos données personnelles.

Services Tiers

Nous n'utilisons que les services Apple :

  • iCloud/CloudKit pour la synchronisation entre appareils
  • StoreKit pour le traitement des achats in-app
  • Face ID/Touch ID pour l'authentification biométrique (optionnel)

Nous n'utilisons pas de services d'analyse, de publicité ni de traceurs tiers.

Sécurité

  • Vos données sont protégées par le chiffrement d'Apple
  • Vous pouvez activer le verrouillage biométrique (Face ID/Touch ID)
  • La synchronisation iCloud utilise le chiffrement de bout en bout

Vos Droits

Vous avez un contrôle total sur vos données :

  • Accès - Vous pouvez consulter toutes vos données dans l'application
  • Exportation - Vous pouvez exporter vos données à tout moment
  • Suppression - Vous pouvez supprimer n'importe quelle donnée ou désinstaller l'application
  • Portabilité - Vos données sont dans votre iCloud et accessibles depuis n'importe quel appareil Apple
